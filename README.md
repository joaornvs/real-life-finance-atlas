# 🎯 Objetivo do Projeto

O objetivo deste projeto é criar um **mini guia estruturado de educação financeira assistido por IA**, que ajude iniciantes a compreender conceitos fundamentais e aplicá-los em situações reais.

Este atlas financeiro reúne:

- 📚 resumos estruturados de conceitos financeiros  
- 🧠 prompts reutilizáveis para estudo com IA  
- 📖 glossário de termos financeiros  
- 🌍 cenários financeiros da vida real  
- 🤖 respostas geradas por IA com base em fontes abertas  

O projeto também incentiva a **aprendizagem ativa com inteligência artificial**, promovendo pensamento crítico, curadoria de fontes e organização do conhecimento.
